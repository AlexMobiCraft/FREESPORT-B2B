# Обзор проекта FREESPORT (Project Overview)

FREESPORT — это высокопроизводительная E-commerce платформа, спроектированная для обслуживания как розничных (B2C), так и корпоративных (B2B) клиентов.

## 🎯 Основные цели

- Объединение 5 спортивных брендов в единую торговую площадку.
- Реализация сложной ролевой системы цен (6 уровней).
- Глубокая асинхронная интеграция с 1С ERP.
- Высокая производительность за счет гибридного рендеринга (Next.js SSR/ISR).

## 🏗️ Архитектура системы

Система построена по принципу **Decoupled Monorepo**:

- **Backend**: Мощное API на Django REST Framework. Служит "источником истины" для данных.
- **Frontend**: Современное приложение на Next.js 15. Отвечает за рендеринг и UX.
- **Интеграционный слой**: Celery + Redis для фоновых задач по импорту данных и отправке уведомлений.

## 🔑 Ключевые возможности

1.  **Ролевое управление**: Автоматический расчет цен в каталоге в зависимости от группы пользователя.
2.  **Умная корзина**: Поддержка гостевых сессий с сохранением данных при последующей регистрации.
3.  **Гибридный импорт**: Система импорта изображений и данных из 1С с механизмами дедупликации и нормализации названий.
4.  **Мульти-темность**: Архитектурная поддержка различных визуальных тем (`(blue)`, `(electric)`).

## 🛠️ Стек технологий

- **Backend**: Python 3.11+, Django 5.2, DRF, PostgreSQL, Redis, Celery.
- **Frontend**: TypeScript, Next.js 15, React 19, Tailwind CSS 4, Zustand.
- **DevOps**: Docker, Docker Compose, Nginx, GitHub Actions.
