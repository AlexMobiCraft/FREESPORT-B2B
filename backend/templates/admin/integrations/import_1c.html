🚀 Импорт данных из 1С

Выберите тип данных для импорта из системы 1С. Импорт будет выполнен в фоновом режиме, и вы сможете отслеживать его прогресс на странице "Сессии импорта".

ℹ️ Важная информация: После запуска импорта вы будете автоматически перенаправлены на страницу сессий импорта, где сможете в реальном времени отслеживать прогресс выполнения задачи.
⚠️ Требования к данным: Перед запуском импорта убедитесь, что данные выгружены из 1С в директорию:
data/import_1c/

Требуемая структура директорий:
  • Для полного каталога: goods/, offers/, prices/, rests/, priceLists/
  • Для изображений: goods/import_files/
  • Для остатков: rests/
  • Для цен: prices/
  • Для клиентов: contragents/
